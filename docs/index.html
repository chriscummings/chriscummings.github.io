Demeter

An Arduino & Raspberry Pi-based hydroponics robot.

The software is written in Rust and Python3. It directly administers water level top-off and pH buffer solution while monitoring various additional environmental values. Logs to Redis and AdaFruit.IO. It's currently growing peace lilies.

https://github.com/chriscummings/demeter
